# Чат-боты — Telegram, WhatsApp і Slack

> Кіруйце фінансамі прама з Telegram, WhatsApp або Slack. Размаўляйце з ШІ, дадавайце выдаткі, скануйце чэкі і адпраўляйце галасавыя — без адкрыцця прыкладання.

## Агляд

Падключыце акаўнт да **Telegram**, **WhatsApp**, **Slack** або любой камбінацыі адначасова. Усе тры боты прапануюць аднолькавыя магчымасці — выкарыстоўвайце зручны месэнджэр.

Для падключэння: **Налады → Чат-боты**.

## Прывязка акаўнта

### Telegram
1. Націсніце **Падключыць Telegram** — з'явіцца 6-значны код (дзейны 10 хвілін)
2. Адкрыйце Telegram і знайдзіце бота
3. Адпраўце `/link ВАШ_КОД` (напрыклад, `/link A3F2B1`)
4. Вы ўбачыце «Акаўнт паспяхова прывязаны!»

### WhatsApp
1. Націсніце **Падключыць WhatsApp** — з'явіцца код і QR-код
2. Націсніце **Адкрыць WhatsApp** або адскануйце QR-код
3. Адпраўце `link ВАШ_КОД` боту
4. Вы ўбачыце «Акаўнт паспяхова прывязаны!»

### Slack
1. Націсніце **Падключыць Slack** — з'явіцца 6-значны код (дзейны 10 хвілін)
2. Адкрыйце Slack, знайдзіце праграму **AI Budget Assistant** і адкрыйце асабісты чат з ёй
3. Адпраўце `link ВАШ_КОД` (напрыклад, `link A3F2B1`)
4. Вы ўбачыце «Акаўнт паспяхова прывязаны!»

> Telegram, WhatsApp і Slack можна падключыць адначасова да аднаго акаўнта.

## Каманды

| Каманда | Што робіць |
|---|---|
| `/link КОД` | Прывязаць мэсэнджэр да прыкладання |
| `/expense 50 абед` | Дадаць выдатак |
| `/income 3000 заплата` | Дадаць даход |
| `/usage` | Праверыць выкарыстанне ШІ |
| `/account` | Пераключыць акаўнт |
| `/newchat` | Пачаць новую размову з ШІ |
| `/unlink` | Адключыць бота |
| `/help` | Паказаць усе каманды |

> У **WhatsApp** і **Slack** каманды працуюць з `/` або без яго. Можна таксама проста напісаць суму і апісанне: `50 абед`.
