# Даведнікі

> Катэгорыі, крамы, тэгі і праекты — аснова арганізацыі выдаткаў. Кіруйце імі з аднаго месца: **Налады → Даведнікі**.

## Агляд

Усе чатыры тыпы даведнікаў кіруюцца з адзінага раздзела ў Наладах. Інтэрфейс адзіны: **націсніце на радок для рэдагавання**, **+** для дадання, **іконка кошыка** для выдалення.

> **Роля Назіральнік**: удзельнікі з гэтай роляй могуць праглядаць даведнікі, але не могуць дадаваць, перайменоўваць або выдаляць.

## Катэгорыі

- Націсніце на катэгорыю для перайменавання або змены колеру
- Націсніце **+** для стварэння новай катэгорыі
- Націсніце іконку кошыка для выдалення

## Крамы

Крамы ствараюцца аўтаматычна пры даданні выдаткаў.

- Націсніце на краму для перайменавання
- Перайменаванне аб'ядноўвае ўсе выдаткі пад новым назовам
- Выдаленне прыбірае назву крамы з усіх адпаведных выдаткаў

### Правілы катэгорый

Дадатак вучыцца на вашых выпраўленнях. Кожны раз, калі вы мяняеце катэгорыю расходу з назвай крамы, аўтаматычна захоўваецца **правіла катэгорыі**. Пры наступным імпарце выпіскі або CSV-файла Wise дадатак ужые ваша правіла і прызначыць катэгорыю без ручнога выпраўлення.

- Вывучаныя правілы адлюстроўваюцца ў раздзеле **Правілы катэгорый** ўнізе экрана Крамы
- Націсніце кошык для выдалення правіла
- Правілы захоўваюцца на серверы і сінхранізуюцца на ўсіх прыладах

## Тэгі

- Націсніце **+** для стварэння тэга (назва і колер)
- Націсніце на тэг для рэдагавання
- Націсніце кошык для выдалення тэга

## Праекты

- Націсніце **+** для стварэння праекта (назва, апісанне, колер, неабавязковы бюджэт)
- Націсніце на праект для адкрыцця дэталёвага экрана з выдаткамі
- Выкарыстоўвайце **іконку алоўка** для рэдагавання і **іконку кошыка** для выдалення
