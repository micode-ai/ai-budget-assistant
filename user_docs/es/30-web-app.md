# Aplicación web

> Usa AI Budget Assistant directamente en tu navegador en https://ai-budget.pl — sin instalación. Comparte la misma cuenta y los mismos datos que la app móvil.

## Cómo abrir la aplicación web

1. Abre https://ai-budget.pl en cualquier navegador moderno (Chrome, Safari, Firefox, Edge)
2. Inicia sesión con el **mismo correo y contraseña** que usas en el teléfono
3. Tus cuentas, transacciones y ajustes aparecen automáticamente — se sincronizan desde el mismo servidor

No hay nada que descargar ni instalar. La aplicación web funciona en ordenadores de escritorio, portátiles, tabletas y navegadores de teléfono.

## Qué puedes hacer

La versión web cubre lo esencial del día a día:

- **Ver** tu panel, saldos, gastos e ingresos
- **Añadir y editar** transacciones, presupuestos, categorías y etiquetas
- **Analíticas** — gráficos, desgloses y tendencias
- **Chat con IA** — haz preguntas y ejecuta comandos en lenguaje natural
- **Cuentas** — cambia entre tu cuenta personal y las compartidas, cambia la moneda de visualización

Es ideal para sesiones largas con un teclado de verdad: revisar el mes, ordenar categorías o conversar con el asistente de IA.

## Limitaciones

La versión web es un complemento de la app móvil, no un reemplazo completo. Algunas funciones se comportan de forma distinta en el navegador:

- **Sin modo sin conexión.** La app móvil guarda una copia local de tus datos para funcionar sin conexión. La web **no** — solo muestra lo que devuelve el servidor y necesita conexión a internet. No se almacena nada en tu ordenador.
- **El escaneo de recibos y la entrada por voz están limitados.** La captura con cámara, el OCR de fotos y la grabación de voz están degradados o no disponibles en el navegador. Usa la app móvil para recibos y voz.
- **Sin inicio de sesión biométrico.** El acceso con Face ID / huella es solo para móvil. En la web inicias sesión con tu correo y contraseña cada vez.

## Consejos

- Para tareas grandes, lo mejor es un **navegador de escritorio o portátil**.
- Pulsa **Enter** para enviar en la pantalla de inicio de sesión y en el chat de IA.
- Si los datos parecen desactualizados, actualiza la página — la web siempre lee en directo desde el servidor.
- El inicio de sesión es por sesión del navegador. Actualizar la página te mantiene conectado; cerrar el navegador puede requerir iniciar sesión de nuevo.

## ¿Cuál debo usar?

| Tarea | Mejor en |
|---|---|
| Añadir rápido sobre la marcha | Móvil |
| Escanear recibos, notas de voz | Móvil |
| Trabajar sin conexión | Móvil |
| Revisiones largas, mucha edición con teclado | Web |
| Conversar con la IA con teclado | Web |

Ambas se mantienen sincronizadas, así que usa la que mejor te venga en cada momento.
