# Application web

Utilisez AI Budget Assistant directement dans votre navigateur à l’adresse **https://ai-budget.pl** — sans installation. Elle partage le même compte et les mêmes données que l’application mobile.

## Ouvrir l’application web

1. Ouvrez **https://ai-budget.pl** dans n’importe quel navigateur moderne (Chrome, Safari, Firefox, Edge)
2. Connectez-vous avec les **mêmes e-mail et mot de passe** que sur votre téléphone
3. Vos comptes, transactions et réglages apparaissent automatiquement — ils se synchronisent depuis le même serveur

Rien à télécharger ni à installer. L’application web fonctionne sur ordinateurs de bureau, portables, tablettes et navigateurs de téléphone.

## Ce que vous pouvez faire

La version web couvre l’essentiel du quotidien :

- **Consulter** votre tableau de bord, vos soldes, dépenses et revenus
- **Ajouter et modifier** des transactions, budgets, catégories et étiquettes
- **Analyses** — graphiques, répartitions et tendances
- **Chat IA** — posez des questions et exécutez des commandes en langage naturel
- **Comptes** — basculez entre votre compte personnel et les comptes partagés, changez la devise d’affichage

C’est idéal pour les sessions plus longues avec un vrai clavier : passer le mois en revue, ranger les catégories ou discuter avec l’assistant IA.

## Limitations

La version web complète l’application mobile sans la remplacer entièrement. Quelques fonctions se comportent différemment dans le navigateur :

- **Pas de mode hors ligne.** L’application mobile conserve une copie locale de vos données pour fonctionner sans connexion. La version web **non** — elle n’affiche que ce que renvoie le serveur et nécessite une connexion internet. Rien n’est mis en cache sur votre ordinateur.
- **Le scan de reçus et la saisie vocale sont limités.** La capture par caméra, l’OCR de photos et l’enregistrement vocal sont dégradés ou indisponibles dans le navigateur. Utilisez l’application mobile pour les reçus et la voix.
- **Pas de connexion biométrique.** La connexion par Face ID / empreinte est réservée au mobile. Sur le web, vous vous connectez à chaque fois avec votre e-mail et votre mot de passe.

## Astuces

- Pour les grosses tâches, privilégiez un **navigateur sur ordinateur ou portable**.
- Appuyez sur **Entrée** pour valider sur l’écran de connexion et dans le chat IA.
- Si les données semblent obsolètes, actualisez la page — la version web lit toujours en direct depuis le serveur.
- La connexion vaut pour la session du navigateur. Actualiser la page vous maintient connecté ; fermer le navigateur peut exiger une reconnexion.

## Que choisir ?

| Tâche | Idéal sur |
|---|---|
| Ajout rapide en déplacement | Mobile |
| Scanner des reçus, notes vocales | Mobile |
| Travailler hors ligne | Mobile |
| Longues revues, beaucoup de saisie au clavier | Web |
| Discuter avec l’IA au clavier | Web |

Les deux restent synchronisées : utilisez celle qui convient sur le moment.
