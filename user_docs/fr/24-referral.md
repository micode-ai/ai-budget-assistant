# Inviter des amis — Programme de parrainage

> Invitez des amis, gagnez des requêtes AI bonus et débloquez des objectifs. Partagez votre code de parrainage unique et soyez récompensé lorsque vos amis utilisent activement l'application.

## Aperçu

Le programme de parrainage vous permet d'inviter des amis sur AI Budget Assistant. Lorsqu'ils s'inscrivent avec votre code et utilisent activement l'application pendant 7 jours, vous recevez **+30 requêtes AI bonus**. Les amis parrainés bénéficient d'un **essai prolongé de 14 jours** au lieu des 7 jours standard.

## Votre code de parrainage

Chaque utilisateur possède un code de parrainage unique de 6 caractères. Vous le trouverez sur l'écran **Inviter des amis** :

1. Allez dans **Paramètres** → **Inviter des amis**
2. Votre code est affiché en haut de l'écran
3. Appuyez sur **Copier** pour copier le code
4. Appuyez sur **Partager** pour l'envoyer via une application de messagerie

## Comment ça marche

1. **Partagez votre code** — envoyez-le à vos amis par messagerie ou réseaux sociaux
2. **L'ami s'inscrit** — il saisit votre code lors de l'inscription (champ facultatif)
3. **Période d'attente de 7 jours** — le parrainage reste en statut « En attente »
4. **L'ami utilise l'application** — il doit utiliser activement l'appli (synchroniser les données) pendant 7 jours
5. **Le parrainage devient Actif** — vous recevez **+30 requêtes AI** en bonus

## Statuts des parrainages

| Statut | Signification |
|---|---|
| **En attente** | L'ami s'est inscrit, attente de 7 jours pour confirmation d'activité |
| **Actif** | L'ami confirmé comme utilisateur actif, bonus accordé |
| **Expiré** | 30 jours écoulés sans confirmation d'activité |

## Statistiques

L'écran « Inviter des amis » affiche vos statistiques :

- **Total** — combien de personnes ont utilisé votre code
- **Actifs** — parrainages actifs confirmés (bonus accordé)
- **En attente** — en attente de confirmation d'activité

## Requêtes AI bonus

Pour chaque parrainage au statut **Actif**, vous recevez **+30 requêtes AI**. Les requêtes bonus s'ajoutent à la limite mensuelle de votre plan et ne sont pas réinitialisées chaque mois.

Le total de vos requêtes bonus est affiché sur l'écran « Inviter des amis ».

## Objectifs

En accumulant des parrainages actifs, vous débloquez des objectifs :

| Objectif | Récompense |
|---|---|
| **5 parrainages actifs** | Mois Pro gratuit (code promo envoyé par email) |
| **10 parrainages actifs** | Badge Ambassadeur |

Votre progression vers le prochain objectif est affichée sous forme de barre de progression.

## Avantages pour les amis parrainés

Les amis qui s'inscrivent avec votre code de parrainage reçoivent :

- **Essai prolongé de 14 jours** (au lieu de l'essai standard de 7 jours)
- La date de fin d'essai est affichée sur l'écran **Abonnement**

## FAQ

- **Q : Où trouver mon code de parrainage ?**
  **R :** Allez dans Paramètres → Inviter des amis. Le code est affiché en haut.

- **Q : Pourquoi mon parrainage est toujours « En attente » ?**
  **R :** Les parrainages restent en attente au moins 7 jours. L'ami doit aussi utiliser activement l'application.

- **Q : Quand un parrainage expire-t-il ?**
  **R :** Si l'ami parrainé ne devient pas utilisateur actif dans les 30 jours.

- **Q : Les requêtes AI bonus sont-elles réinitialisées mensuellement ?**
  **R :** Non, les requêtes bonus des parrainages sont permanentes.

- **Q : Puis-je utiliser mon propre code ?**
  **R :** Non, l'auto-parrainage n'est pas autorisé.
