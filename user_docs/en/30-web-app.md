# Web App

Use AI Budget Assistant right in your browser at **https://ai-budget.pl** — no install needed. It shares the same account and data as the mobile app.

## Opening the Web App

1. Open **https://ai-budget.pl** in any modern browser (Chrome, Safari, Firefox, Edge)
2. Sign in with the **same email and password** you use on your phone
3. Your accounts, transactions, and settings appear automatically — they sync from the same server

There is nothing to download or install. The web app works on desktops, laptops, tablets, and phone browsers.

## What You Can Do

The web app covers the everyday essentials:

- **View** your dashboard, balances, expenses, and income
- **Add and edit** transactions, budgets, categories, and tags
- **Analytics** — charts, breakdowns, and trends
- **AI Chat** — ask questions and run commands in natural language
- **Accounts** — switch between your personal and shared accounts, change the display currency

It's a great way to do longer sessions on a real keyboard — reviewing the month, cleaning up categories, or chatting with the AI assistant.

## Limitations

The web version is a companion to the mobile app, not a full replacement. A few features behave differently in the browser:

- **No offline mode.** The mobile app keeps a local copy of your data so it works without a connection. The web app does **not** — it shows only what the server returns and needs an internet connection. Nothing is cached on your computer.
- **Receipt scanning and voice input are limited.** Camera capture, photo OCR, and voice recording are degraded or unavailable in the browser. Use the mobile app for receipts and voice.
- **No biometric login.** Face ID / fingerprint sign-in is mobile-only. On the web you log in with your email and password each time.

## Tips

- For the best experience on big tasks, use a **desktop or laptop browser**.
- Press **Enter** to submit on the login screen and in AI chat.
- If your data looks out of date, refresh the page — the web app always reads live from the server.
- Your login is per-session in the browser. Refreshing keeps you signed in; closing the browser may require signing in again.

## Which Should I Use?

| Task | Best on |
|---|---|
| Quick add on the go | Mobile |
| Scanning receipts, voice notes | Mobile |
| Working offline | Mobile |
| Long review sessions, typing-heavy edits | Web |
| Chatting with the AI on a keyboard | Web |

Both stay in sync, so use whichever fits the moment.
