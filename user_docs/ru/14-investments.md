# Инвестиционный портфель

> Отслеживайте инвестиционный портфель с актуальными рыночными ценами. Мониторьте акции, ETF, криптовалюты, облигации и товары в одном месте.

## Обзор

Функция инвестиционного портфеля позволяет:

- Отслеживать позиции по разным типам активов
- Видеть цены в реальном времени и стоимость портфеля
- Анализировать доходность за разные периоды
- Сравнивать свои результаты с рыночными бенчмарками (SPY, QQQ, DIA, IWM)
- Записывать сделки покупки/продажи с комиссиями

## Создание инвестиционного счёта

Для отслеживания инвестиций требуется специальный тип счёта **Инвестиции**:

1. Перейдите на вкладку **Счета**
2. Нажмите **Создать счёт**
3. Выберите тип **Инвестиции**
4. Назовите портфель (например, "Основной портфель", "Пенсионный")
5. Нажмите **Создать**

## Добавление позиций

### Поиск активов

1. Откройте инвестиционный счёт
2. Нажмите **Добавить позицию**
3. Ищите по тикеру (например, "AAPL") или названию компании (например, "Apple")
4. Выберите нужный актив из результатов поиска
5. Добавьте заметки (опционально)
6. Нажмите **Сохранить**

### Поддерживаемые типы активов

| Тип | Примеры |
|-----|---------|
| Акции | AAPL, MSFT, GOOGL |
| ETF | SPY, QQQ, VTI |
| Криптовалюты | BTC, ETH, SOL |
| Облигации | Государственные и корпоративные |
| Товары | Золото, серебро, нефть |

## Запись транзакций

После добавления позиции записывайте сделки покупки/продажи:

1. Откройте детали позиции
2. Нажмите **Добавить транзакцию**
3. Выберите тип: **Покупка** или **Продажа**
4. Введите:
   - **Количество** — число акций/единиц
   - **Цена за единицу** — цена покупки/продажи
   - **Комиссия** — брокерская комиссия (опционально)
   - **Дата** — дата сделки
   - **Заметки** — дополнительная информация (опционально)
5. Нажмите **Сохранить**

Приложение автоматически рассчитывает:
- **Средняя цена покупки** — средневзвешенная цена приобретения
- **Сумма инвестиций** — сумма всех покупок минус продажи
- **Текущая прибыль/убыток** — на основе текущей цены

## Сводка по портфелю

Главный экран инвестиций показывает:

- **Общая стоимость** — текущая рыночная стоимость всех позиций
- **Общая прибыль/убыток** — сумма прибыли или убытка
- **Общая доходность %** — процентный возврат
- **Изменение за день** — сегодняшнее изменение стоимости

Для каждой позиции отображается:
- Текущая цена и дневное изменение
- Ваше количество и средняя цена
- Индивидуальная прибыль/убыток и доля в портфеле

## Аналитика

Доступ к детальной аналитике портфеля:

1. Нажмите кнопку **Аналитика**
2. Выберите период: 1Н, 1М, 3М, 1Г или Всё время

### График производительности

Показывает стоимость портфеля во времени по сравнению с инвестированной суммой. Область между линиями отображает вашу прибыль или убыток.

### Распределение по типам

Визуализирует распределение портфеля по типам активов (акции, ETF, крипто и т.д.).

### Лидеры роста и падения

Список лучших и худших позиций по процентной доходности.

### AI-инсайты портфеля (Pro+)

Получайте AI-анализ вашего портфеля с рекомендациями к действию:

1. Откройте вкладку **Аналитика**
2. Прокрутите к карусели **Инсайты** вверху
3. Свайпайте влево/вправо для просмотра разных инсайтов
4. Нажмите кнопку закрытия, чтобы скрыть инсайт

**Типы инсайтов:**

| Тип | Описание |
|-----|----------|
| Риск концентрации | Предупреждает, когда один актив доминирует в портфеле |
| Дисбаланс секторов | Сигнализирует о перевесе в одном типе активов |
| Отстающий | Выявляет активы, отстающие от рынка |
| Лидер роста | Подсвечивает возможности для ребалансировки |
| Отклонение от бенчмарка | Показывает отклонение портфеля от бенчмарка |
| Пробел диверсификации | Предлагает добавить недостающие типы активов |
| Налоговый сигнал | Уведомляет о существенных нереализованных П/У |
| Влияние комиссий | Предупреждает, когда комиссии съедают прибыль |

Каждый инсайт включает визуализацию (график) и рекомендацию к действию.

**Примечание:** AI-инсайты кэшируются на 24 часа, стоимость обновления — 2.5 AI-кредитов.

### Сравнение с бенчмарком (Pro+)

Сравните доходность портфеля с рыночными индексами:

| Бенчмарк | Описание |
|----------|----------|
| SPY | Индекс S&P 500 |
| QQQ | Индекс Nasdaq 100 |
| DIA | Индекс Dow Jones Industrial |
| IWM | Russell 2000 (малая капитализация) |

**Понимание сравнения:**
- **Доходность портфеля** — ваша фактическая прибыль/убыток в процентах
- **Доходность бенчмарка** — производительность индекса за тот же период
- **Разница** — насколько вы превзошли или отстали от рынка

## Понимание расчётов

Нажмите на любую карточку аналитики, чтобы увидеть объяснение формулы:

### Доходность
```
Доходность % = ((Конечная стоимость - Начальная стоимость) / Начальная стоимость) x 100
```

### Прибыль/убыток (П/У)
```
П/У = Текущая стоимость - Сумма инвестиций
П/У % = (П/У / Сумма инвестиций) x 100
```

### Распределение
```
Доля % = (Стоимость актива / Общая стоимость портфеля) x 100
```

## Обновление цен

- Цены обновляются автоматически каждые 15 минут
- Нажмите кнопку **Обновить** для немедленного обновления
- Исторические цены кэшируются для экономии трафика

## Советы

1. **Диверсифицируйте учёт** — добавьте все инвестиции для полной картины
2. **Учитывайте комиссии** — включайте брокерские сборы для точного расчёта прибыли
3. **Используйте бенчмарки** — сравнивайте с индексами для оценки результатов
4. **Регулярно проверяйте** — смотрите аналитику еженедельно для выявления трендов

## Ограничения

- Данные о ценах предоставляются через Twelve Data API
- Некоторые экзотические инструменты могут быть недоступны
- Исторические данные ограничены торговыми днями
- Цены в реальном времени могут иметь задержку до 15 минут
