# Аналитика

> Визуализируйте структуру расходов с помощью интерактивных графиков. Фильтруйте по периоду времени и валюте, детализируйте данные и получайте ИИ-инсайты для лучшего понимания своих финансов.

## Обзор

Вкладка **Аналитика** предоставляет полную картину ваших расходов через графики, инсайты и исследование данных. Все данные основаны на расходах в выбранном счёте.

## Временной диапазон и фильтры

В верхней части экрана Аналитики:

- **Переключатель периода**: **Неделя** | **Месяц** | **Год**
- **Фильтр по валюте**: **Все** валюты или выбрать конкретную (USD, EUR, PLN и т.д.)

Выбранный диапазон влияет на все графики и инсайты ниже.

## Сводные карточки

- **Всего потрачено** — общая сумма расходов за выбранный период
- **Сравнение тренда** — «больше, чем за прошлый месяц/неделю/год» или «меньше, чем за прошлый месяц/неделю/год» с процентами
- **Среднее в день** — среднедневные расходы

## Карусель ИИ-инсайтов

Горизонтально прокручиваемая карусель ИИ-генерируемых инсайтов (требуется план Pro):

- **Аномалии расходов** — обнаружены необычные расходы (например, «78% больше обычного на Транспорт»)
- **Прогнозы бюджета** — когда ваш бюджет предположительно закончится
- **Возможность экономии** — предложения по сокращению расходов
- **Сравнение категорий** — как ваши расходы распределяются между категориями
- **Изменение тренда** — значительные сдвиги в структуре расходов

Каждая карточка инсайта имеет уровень важности: критический (красный), предупреждение (жёлтый) или информационный (синий).

> **Примечание:** Пользователи бесплатного плана видят предложение улучшить тариф: «Перейдите на Pro для ИИ-инсайтов».

## Графики

### Динамика расходов (Столбчатый график)

- Показывает ежедневные или ежемесячные расходы за выбранный период
- Интерактивный: нажмите на любой столбец для детализации этого временного сегмента

### Расходы по категориям (Кольцевая диаграмма)

- Разбивка по категориям с процентами
- Цветовая маркировка по категориям
- Нажмите на сегмент, чтобы изучить расходы этой категории

### Бюджет vs Факт (Группированный столбчатый график)

- Сравнение лимитов бюджета и фактических расходов
- Показывает **В норме** или **Превышен** по каждой категории
- Отображается только при наличии активных бюджетов

### Расходы по дням недели (График по дням)

- Анализ паттернов: в какие дни вы тратите больше всего
- Инсайт: «Больше всего вы тратите по субботам»

### По тегам / По проектам

- **Разбивка по тегам** — расходы, сгруппированные по вашим пользовательским тегам
- **Сравнение проектов** — расходы, сгруппированные по проектам
- Помогает отслеживать тематические расходы (например, все расходы #кофе или затраты проекта «Отпуск»)

## Детализация

Нажмите на любой элемент графика для более глубокого изучения:

1. Вид **Год** — нажмите на столбец месяца, чтобы увеличить этот месяц
2. Вид **Месяц** — нажмите на неделю, чтобы увеличить эту неделю
3. Вид **Неделя** — нажмите на день, чтобы увидеть дневные транзакции
4. Вид **День** — просматривайте отдельные транзакции

Используйте кнопку **Назад**, чтобы вернуться на уровень выше.

## Краткие выводы

Под графиками вы найдёте текстовые краткие инсайты:

- **Топ категория** — ваша категория с наибольшими расходами за этот период
- **Пиковый день расходов** — день с наибольшими расходами
- **Совет по дневному бюджету** — рекомендуемые ежедневные расходы, чтобы уложиться в бюджет
- **Экономия по чекам** — общая сумма экономии по скидкам

Нажмите **Экспорт отчёта**, чтобы перейти на экран [Экспорт и отчёты](./16-export-reports.md), где можно генерировать PDF, Excel и CSV отчёты, просматривать ежемесячные дайджесты и управлять резервными копиями данных.

## Часто задаваемые вопросы

- **В: Почему я не вижу графиков?**
  **О:** Графики появляются после того, как у вас есть данные о расходах. Сначала добавьте расходы, затем проверьте Аналитику.

- **В: Как генерируются ИИ-инсайты?**
  **О:** Инсайты генерируются путём анализа ваших паттернов расходов, сравнения с историческими данными и обнаружения аномалий. Для этого требуется подписка Pro или Business.
