# Telegram Бот

> Управляйте своими финансами прямо из Telegram. Общайтесь с ИИ, добавляйте расходы командами, сканируйте чеки и отправляйте голосовые сообщения — всё без открытия приложения.

## Обзор

**Telegram Бот** позволяет взаимодействовать с вашим ИИ-помощником по бюджету прямо из Telegram. Привяжите аккаунт один раз — и вы сможете отслеживать расходы, задавать финансовые вопросы и управлять бюджетами прямо из мессенджера.

## Привязка аккаунта

1. Откройте приложение и перейдите в **Настройки**
2. Нажмите **Telegram Бот** в разделе Интеграции
3. Нажмите **Сгенерировать код** — появится 6-значный код (действителен 10 минут)
4. Откройте Telegram и найдите бота
5. Отправьте `/link КОД` (например, `/link A3F2B1`)
6. Вы увидите подтверждение: «Аккаунт успешно привязан!»

> **Примечание:** Каждый аккаунт Telegram может быть привязан только к одному аккаунту приложения. Повторная привязка заменяет предыдущую.

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветственное сообщение и инструкции |
| `/link КОД` | Привязать Telegram к приложению |
| `/expense СУММА ОПИСАНИЕ` | Быстро добавить расход (например, `/expense 50 обед`) |
| `/income СУММА ОПИСАНИЕ` | Быстро добавить доход (например, `/income 3000 зарплата`) |
| `/category [ТИП] НАЗВАНИЕ` | Создать категорию (например, `/category expense Еда`) |
| `/categories` | Список и удаление категорий |
| `/account` | Переключиться между счетами |
| `/newchat` | Начать новый разговор с ИИ |
| `/unlink` | Отвязать Telegram от аккаунта |
| `/help` | Показать все доступные команды |

## ИИ Чат в Telegram

Отправьте любое текстовое сообщение боту, и оно будет обработано ИИ-помощником — тем же, что доступен во вкладке ИИ Чат в приложении.

**Примеры:**
- «На что я потратил больше всего в этом месяце?»
- «Покажи мои расходы за прошлую неделю»
- «Добавь расход 500₴ на продукты»
- «Какой статус моего бюджета?»

ИИ поддерживает все функции из чата в приложении: команды на естественном языке, подтверждение действий, разбивку по категориям и анализ бюджета.

## Автоматическое определение счёта

Если у вас несколько счетов (например, «Личный» и «Семейный»), ИИ автоматически определяет, когда вы упоминаете название счёта в сообщении, и запрашивает данные из нужного счёта.

**Примеры:**
- «Покажи мои расходы в Семейном счёте» — запросит данные из Семейного счёта
- «Сколько я потратил на еду?» — запросит данные из счёта по умолчанию
- «Добавь расход 100₴ на продукты в Семейный» — создаст расход в Семейном счёте

> **Примечание:** Это не меняет ваш счёт по умолчанию. Используйте `/account` для переключения.

## Голосовые сообщения

1. Запишите голосовое сообщение в Telegram
2. Отправьте его боту
3. Бот расшифрует вашу речь и обработает как сообщение ИИ-чата

Голосовые сообщения поддерживают все те же команды и вопросы, что и текстовые.

## Сканирование чеков

1. Сфотографируйте чек
2. Отправьте фото боту
3. Бот сканирует его с помощью OCR и покажет сводку
4. Нажмите **Подтвердить**, чтобы добавить расход, или **Отменить**, чтобы отклонить

Вы также можете отправлять изображения чеков как документы (PDF или изображения).

## Переключение счетов

Если у вас несколько счетов:

1. Отправьте `/account`
2. Бот покажет все ваши счета с инлайн-кнопками
3. Нажмите на счёт, на который хотите переключиться
4. Активный счёт отмечен галочкой

Все последующие команды и запросы ИИ будут использовать выбранный счёт, пока вы не переключитесь снова.

## Поддержка валют

Бот распознаёт символы и коды валют в командах:

| Символ | Валюта |
|---|---|
| ₴ | UAH |
| $ | USD |
| € | EUR |
| zł | PLN |
| £ | GBP |
| ₽ | RUB |

**Примеры:** `/expense 50$ обед`, `/expense 100₴ продукты`, `/expense 30 EUR такси`

## Часто задаваемые вопросы

- **В: Можно ли использовать бота без привязки?**
  **О:** Нет, сначала нужно привязать аккаунт Telegram с помощью кода из приложения.

- **В: Работает ли бот в групповых чатах?**
  **О:** Бот предназначен только для личных (1:1) переписок.

- **В: Какой счёт использует бот?**
  **О:** Бот использует ваш счёт по умолчанию (установленный при привязке или через `/account`). Вы также можете упомянуть название счёта в сообщении, и ИИ автоматически использует этот счёт для запроса.

- **В: Можно ли привязать несколько аккаунтов Telegram?**
  **О:** Нет, у каждого пользователя приложения может быть один привязанный аккаунт Telegram, и каждый аккаунт Telegram может быть привязан к одному пользователю.

- **В: Сообщения боту учитываются в лимите запросов ИИ?**
  **О:** Да, каждое обработанное ИИ сообщение (текстовое, голосовое) расходует один запрос из вашего ежемесячного лимита.
