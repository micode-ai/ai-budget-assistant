# Импорт транзакций из банка

> Импортируйте транзакции напрямую из CSV-экспортов крупнейших польских банков или любого другого банка с помощью универсального маппера столбцов.

## Поддерживаемые банки

Вы можете импортировать транзакции напрямую из CSV-экспортов крупнейших польских банков: **mBank, PKO BP, ING Bank Śląski, Bank Millennium, Pekao SA**. Для любого другого банка универсальный маппер столбцов позволяет описать формат вручную.

## Как импортировать

1. Перейдите в **Настройки → Импорт транзакций**
2. Выберите свой банк из списка (или «Другой (пользовательский CSV)» для неподдерживаемых банков)
3. Выберите CSV-файл, экспортированный из интернет-банка
4. Приложение покажет предварительный просмотр, где каждая строка помечена как расход, доход или обмен валюты
5. Снимите отметки с ненужных строк и нажмите **Импортировать**

Приложение запоминает уже импортированные строки по дате, сумме и описанию — повторная загрузка того же CSV не создаст дубликатов.

## Где найти CSV в своём банке

- **mBank**: Интернет-банк → Historia operacji → Eksport → CSV
- **PKO BP**: iPKO → Lista operacji → Pobierz → CSV
- **ING Bank Śląski**: Moje ING → Historia → Eksportuj → CSV
- **Bank Millennium**: Web → Historia rachunku → Eksport → CSV
- **Pekao SA**: Pekao24 → Historia → Eksport → CSV

## Что импортируется

Каждая строка становится Расходом, Доходом или Обменом валюты (когда приложение обнаруживает парную FX-транзакцию с той же датой в разных валютах). Категории предлагаются автоматически для популярных торговых сетей (Biedronka, Żabka, Orlen, Lidl и др.) — их можно изменить позднее.

## «Другой» — универсальный маппер CSV

Если вашего банка нет в списке, выберите «Другой (пользовательский CSV)». Приложение покажет предварительный просмотр файла и попросит указать, в каком столбце находится дата, сумма и описание. Можно сохранить это сопоставление с именем — следующий CSV с той же структурой столбцов будет импортирован автоматически.

## Кодировка

Приложение автоматически определяет UTF-8 и Windows-1250 (наиболее распространённую кодировку польских банков). Если в предварительном просмотре польские символы отображаются некорректно, выберите кодировку вручную в маппере.
