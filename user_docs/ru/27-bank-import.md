# Импорт транзакций из банка

> Импортируйте транзакции из CSV или PDF-выписки вашего банка. Поддерживаются mBank, PKO BP, Erste Bank, Alior Bank, Revolut, Wise и любой другой банк через универсальный маппер колонок.

## Поддерживаемые банки

- **mBank** — экспорт CSV
- **PKO BP** — экспорт CSV
- **Erste Bank** — PDF-выписка
- **Alior Bank** — PDF-выписка
- **Revolut** — экспорт CSV
- **Wise** — экспорт CSV (мультивалютный, конвертации определяются автоматически)
- **Другой** — любой банк, через универсальный маппер колонок (CSV)

## Как импортировать

1. Перейдите в **Настройки → Импорт транзакций**
2. Выберите ваш банк из списка (или **Другой (CSV)**, если он не в списке)
3. Выберите файл, экспортированный из банка
4. Приложение покажет предпросмотр — каждая строка помечена как расход, доход или обмен валюты
5. Снимите галочки с ненужных строк и нажмите **Импортировать**

Приложение пропускает строки, которые уже есть в аккаунте, сопоставляя по дате, сумме и валюте.

## Где найти экспорт в вашем банке

- **mBank**: Веб-банкинг → Historia operacji → Eksport → CSV
- **PKO BP**: iPKO → Historia operacji → Eksportuj → CSV
- **Erste Bank**: bankowość internetowa → Wyciągi → pobierz wyciąg (PDF)
- **Alior Bank**: Alior Online → Wyciągi → pobierz wyciąg (PDF)
- **Revolut**: приложение Revolut → Statements → выберите диапазон дат → CSV → Download
- **Wise**: wise.com → Transactions → Statements and Reports → выберите диапазон дат → CSV → выберите валюту → Download

> **Совет для Wise:** Wise создаёт один CSV на каждый валютный баланс. Импортируйте каждую валюту отдельно. До 469 дней за один экспорт.

## Wise — конвертации и комиссии

При конвертации валют внутри Wise (например, 100 USD → EUR) создаётся две строки. Приложение определяет такие пары автоматически и создаёт одну запись **Обмен валюты** (раздел Кошелёк → Обмены).

Комиссии Wise из столбца `Total fees` автоматически включаются в сумму расхода.

## Что импортируется

Каждая строка становится Расходом, Доходом или Обменом валюты. Категории предлагаются автоматически для популярных магазинов. Каждая строка помечается уникальным ID — повторный импорт того же файла безопасен.

## «Другой» — универсальный маппер

Если вашего банка нет в списке, выберите **Другой (CSV)**. Приложение покажет предпросмотр файла и попросит указать, в каком столбце находятся дата, сумма и описание. Сохраните маппинг — следующий CSV с той же структурой импортируется автоматически.

## История импортов и Отмена

Раздел **Прошлые импорты** в нижней части **Настройки → Импорт транзакций** показывает последние 20 импортов.

Чтобы отменить импорт, нажмите **стрелку отмены** (↩) справа. Все транзакции из этого импорта будут удалены, блокировка дубликатов снята.

- Отмена доступна в течение **30 дней** с момента импорта.

## Не видите свой банк?

В нижней части **Настройки → Импорт транзакций** есть карточка **«Нет вашего банка?»**. Нажмите, укажите название банка и прикрепите пример выписки. Запрос сразу попадёт к нашей команде.

## Кодировка

Для CSV-файлов автоматически определяются UTF-8 и Windows-1250. PDF-выписки читаются напрямую.
