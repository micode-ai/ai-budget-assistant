# Импорт транзакций из банка

> Импортируйте транзакции из CSV-файла или PDF-выписки вашего банка либо из любого другого банка с помощью универсального маппера столбцов.

## Поддерживаемые банки

- **mBank** — экспорт CSV
- **PKO BP** — экспорт CSV
- **Erste Bank** — PDF-выписка
- **Alior Bank** — PDF-выписка
- **Wise** — экспорт CSV (см. [Импорт из Wise](./26-wise-import.md))
- **Other** — любой банк, через универсальный маппер столбцов (CSV)

Поддержка новых банков добавляется постепенно. Если вашего банка ещё нет в списке, выберите **Other** и вручную укажите соответствие столбцов.

## Как импортировать

1. Перейдите в **Настройки → Импорт транзакций**
2. Выберите свой банк из списка (или **Other (custom CSV)**, если его нет в списке)
3. Выберите файл, экспортированный из вашего банка — **CSV** для mBank/PKO, **PDF**-выписку для Erste/Alior
4. Приложение покажет предварительный просмотр, где каждая строка помечена как расход, доход или обмен валюты
5. Снимите отметки с ненужных строк и нажмите **Импортировать**

Приложение пропускает строки, которые уже есть в вашем счёте — будь то ранее импортированные или добавленные вручную — сверяя их по дате, сумме и валюте, поэтому повторный импорт не создаёт дублей. Совпавшие строки по умолчанию сняты с отметки; включите одну из них, если это действительно отдельная транзакция.

## Где найти файл в своём банке

- **mBank**: Интернет-банк → Historia operacji → Eksport → CSV
- **PKO BP**: iPKO → Historia operacji → Eksportuj → CSV
- **Erste Bank**: bankowość internetowa → Wyciągi → pobierz wyciąg (PDF)
- **Alior Bank**: Alior Online → Wyciągi → pobierz wyciąg (PDF)

## Что импортируется

Каждая строка становится Расходом, Доходом или Обменом валюты (когда приложение обнаруживает парную FX-транзакцию с той же датой в разных валютах). Категории предлагаются автоматически для популярных торговых сетей (Biedronka, Żabka, Orlen, Lidl, Rossmann и др.) — их можно изменить позднее.

## «Other» — универсальный маппер CSV

Если вашего банка нет в списке, выберите **Other (custom CSV)**. Приложение покажет предварительный просмотр файла и попросит указать, в каком столбце находятся дата, сумма и описание. Можно сохранить это сопоставление с именем — следующий CSV с той же структурой столбцов будет импортирован автоматически.

## Кодировка

Для CSV-файлов приложение автоматически определяет UTF-8 и Windows-1250 (наиболее распространённую кодировку польских банков). Если в предварительном просмотре польские символы отображаются некорректно, выберите кодировку вручную в маппере. PDF-выписки читаются напрямую — выбор кодировки не требуется.
