# Цели накопления

> Задайте финансовую цель, и ИИ составит персональный план — с ежемесячными взносами, лимитами расходов по категориям и контрольными точками, чтобы вы не сбились с пути.

## Обзор

Функция **Цели накопления** позволяет задать цель, например «Накопить 20 000 PLN к следующему лету». ИИ анализирует ваши доходы, расходы и привычки за последние 3 месяца, а затем составляет пошаговый план достижения цели.

## Как открыть

- Перейдите в раздел **Цели** с главного экрана или из настроек
- В списке целей отображаются все ваши активные, приостановленные, выполненные и невыполненные цели

## Создание цели

1. Нажмите кнопку **+** на экране целей
2. Заполните форму:
   - **Название цели** — например, «Фонд отпуска», «Подушка безопасности»
   - **Целевая сумма** — сумма, которую вы хотите накопить
   - **Валюта** — выберите из USD, EUR, PLN, GBP, UAH, RUB, BYN
   - **Срок** — укажите целевую дату или используйте кнопки быстрого выбора: 3 месяца, 6 месяцев, 1 год, 2 года
3. Нажмите **Создать цель**

ИИ автоматически сгенерирует план накопления после создания. Это использует **2 ИИ-запроса** из вашего месячного лимита.

## План ИИ

После создания цели ИИ генерирует подробный план, включающий:

### Суммы взносов
- **Ежемесячный взнос** — сколько откладывать в месяц
- **Еженедельный взнос** — разбивка по неделям

### Оценка выполнимости

ИИ оценивает, насколько реалистична ваша цель:

| Оценка | Значение |
|---|---|
| **Легко** | Достижимо с минимальными изменениями образа жизни |
| **Умеренно** | Потребуется некоторая корректировка расходов |
| **Сложно** | Потребуется значительное сокращение текущих расходов |
| **Нереалистично** | Маловероятно без серьёзных изменений дохода/расходов |

### Лимиты расходов по категориям

Таблица с данными по каждой категории расходов:
- **Текущие** — ваши средние ежемесячные траты
- **Рекомендуемые** — рекомендованный ИИ лимит
- **Экономия** — сколько вы сэкономите в месяц

### Контрольные точки

Визуальная шкала с 3–5 этапами:
- Каждая точка имеет **дату**, **целевую сумму** и **описание**
- Достигнутые точки отмечены зелёной галочкой
- Помогает отслеживать, идёте ли вы по графику

### Резюме

Краткое текстовое пояснение плана с практическими рекомендациями, основанными на ваших привычках расходования.

## Отслеживание прогресса

На экране деталей цели отображается:

- **Прогресс-бар** — процент от целевой суммы
- **Индикатор «По графику» / «Отставание»**
- **Прогнозируемая дата завершения** — когда вы достигнете цели при текущем темпе
- **Ежемесячно необходимо** — сколько ещё нужно откладывать в месяц
- **Отставание на** — на сколько вы отстаёте от графика (если применимо)

## История пополнений

Каждый раз, когда вы нажимаете **Внести сумму**, приложение записывает пополнение как отдельную запись в истории. На экране деталей цели отображается раздел **История пополнений** под прогресс-баром:

- Сумма пополнения и валюта цели
- Дата каждого пополнения
- Пометка для пополнений через AI-чат («AI update»)

История начинается с первого использования «Внести сумму». Предыдущие ручные правки не учитываются ретроспективно. Отображается до 20 последних пополнений.

## Управление целями

- **Обновить сумму** — скорректируйте текущую сумму накоплений
- **Редактировать цель** — измените название, целевую сумму или срок
- **Приостановить / Возобновить** — временно приостановите цель
- **Пересоздать план** — получите новый план ИИ на основе актуальных данных о расходах (стоит 2 ИИ-запроса)
- **Удалить** — безвозвратно удалить цель

## Статусы целей

| Статус | Значение |
|---|---|
| **Активна** | Цель в процессе выполнения |
| **Приостановлена** | Временно приостановлена |
| **Выполнена** | Целевая сумма достигнута |
| **Не выполнена** | Срок истёк, цель не достигнута |

## Стоимость ИИ

| Действие | ИИ-запросы |
|---|---|
| Создание цели (включая генерацию плана) | 2 |
| Пересоздание плана | 2 |
| Просмотр целей / прогресса | Бесплатно |

## Часто задаваемые вопросы

- **В: ИИ обновляет план автоматически?**
  **О:** Нет. План генерируется один раз при создании цели. Нажмите **Пересоздать план**, чтобы получить обновлённый план на основе последних данных о расходах.

- **В: Можно ли создать несколько целей одновременно?**
  **О:** Да, вы можете создавать сколько угодно целей. Каждая цель отслеживается независимо.

- **В: Как ИИ узнаёт о моих привычках расходования?**
  **О:** Он анализирует историю ваших расходов и доходов за последние 3 месяца в текущем выбранном счёте.

- **В: Данные о моей цели видны другим?**
  **О:** Цели привязаны к вашему счёту. В общих счетах другие участники с доступом могут видеть цели.
