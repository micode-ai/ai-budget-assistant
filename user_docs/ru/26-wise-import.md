# Импорт из Wise

> Перенесите всю историю транзакций Wise в приложение за один шаг. Загрузите CSV-выписку — и приложение само создаст соответствующие расходы, доходы и конвертации валют.

## Обзор

Если вы пользуетесь Wise, **Импорт из Wise** позволяет загрузить целую выписку в ваш аккаунт одним нажатием. Больше не нужно вводить транзакции по одной — просто скачайте CSV из Wise, передайте файл приложению и проверьте, что будет создано, перед подтверждением.

Импорт охватывает три типа записей:

- **Расходы** — деньги, ушедшие с вашего баланса Wise (списания)
- **Доходы** — поступления (зачисления)
- **Конвертации валют** — когда вы обменивали между балансами внутри Wise (например, USD → EUR)

Каждая импортированная транзакция помечается, чтобы приложение знало, что она пришла из Wise — если вы загрузите ту же выписку дважды, дубликаты будут обнаружены и пропущены автоматически.

## Шаг 1 — Экспортируйте CSV из Wise

1. Откройте Wise (веб-приложение на **wise.com** или мобильное приложение Wise).
2. Перейдите в **Transactions → Statements and Reports**.
3. Выберите **период** (до 469 дней на один файл).
4. Выберите формат **CSV** и нужную валюту / баланс.
5. Скачайте файл на телефон.

> **Совет:** Wise создаёт по одному CSV на валюту. Если хотите импортировать несколько валют, повторите экспорт для каждой и импортируйте их по очереди.

## Шаг 2 — Импортируйте в приложении

1. Откройте приложение и перейдите в **Настройки → Импорт из Wise**.
2. Нажмите **Выбрать CSV-файл** и укажите скачанный файл.
3. Приложение обработает файл (обычно меньше секунды) и покажет предпросмотр.

## Шаг 3 — Проверьте и подтвердите

В предпросмотре каждая транзакция из CSV показана с флажком.

- **Расходы** отображаются с красной стрелкой вниз, **доходы** — с зелёной стрелкой вверх, **конвертации валют** — со значком обмена и обеими сторонами операции (например, `120.00 USD → 109.50 EUR`).
- Рядом с известными продавцами (Uber, Bolt, Lidl, Starbucks, Amazon, Netflix и т. д.) появляется маленький чип **предлагаемой категории**. Если в активном аккаунте уже есть категория с таким же названием, она будет привязана автоматически.
- Строки, которые вы уже импортировали ранее, **затемнены и помечены «Уже импортировано»** — выбрать их повторно нельзя, что и защищает вас от дубликатов.
- Снимите галочки с того, что не хотите импортировать (например, переводы между своими счетами).

Когда выбор готов, нажмите **Импортировать N записей**. Приложение запишет всё в одну операцию — либо будут созданы все выбранные строки, либо ни одна.

## Что попадает в записи

| Поле | Откуда берётся |
|---|---|
| Дата | Колонка `Date` |
| Сумма | `Amount` (по модулю) + учтена `Total fees` |
| Валюта | Колонка `Currency` |
| Описание | `Description`, или `Merchant`, или `Payment Reference` |
| Категория | Предложена по продавцу, если он распознан; иначе пусто |
| Источник | Помечено как `import`, чтобы фильтровать в аналитике |

## Конвертации валют

Когда один и тот же перевод Wise задействует две валюты (например, вы конвертируете 100 USD в евро), Wise выдаёт две строки — списание в USD и зачисление в EUR. Приложение распознаёт такие пары по общей `Payment Reference` и создаёт единую запись **Конвертация валюты** вместо двух отдельных транзакций. Конвертация появится в **Кошелёк → Обмены** с корректным курсом.

## Повторный импорт

Повторная загрузка того же CSV безопасна. Каждая строка несёт свой `TransferWise ID` от Wise, и приложение не создаст вторую запись для уже импортированного ID. Это значит:

- Можно повторно экспортировать более широкий период и загрузить его — будут созданы только новые строки.
- Можно прервать предпросмотр на полпути и начать заново позже — уже подтверждённые строки запоминаются.

## FAQ

- **В: Это работает с другими банками?**
  **О:** Сейчас поддерживаются только CSV-экспорты Wise. У других банков могут быть иные колонки. Откройте запрос на функциональность, если хотите видеть другой банк.

- **В: Можно импортировать выписку в PDF или XLSX?**
  **О:** Пока нет. Экспортируйте выписки Wise в формате CSV.

- **В: Файл где-то сохраняется так, чтобы об этом стоило беспокоиться?**
  **О:** CSV отправляется на сервер AI Budget Assistant, обрабатывается в памяти и удаляется сразу после генерации предпросмотра. Сохраняются только структурированные строки, которые вы подтвердили, — не сам файл.

- **В: Что происходит с комиссиями, удержанными Wise?**
  **О:** Wise указывает комиссии в отдельной колонке `Total fees`. Приложение прибавляет их к тому же расходу, чтобы итог соответствовал тому, что реально ушло с баланса.

- **В: Я импортировал не те строки — можно откатить?**
  **О:** Да. Импортированные строки — это обычные расходы/доходы — откройте каждую и удалите, как любую другую транзакцию. После удаления вы сможете повторно импортировать ту же строку позже.

- **В: В моём CSV нет заголовков / другой формат. Что делать?**
  **О:** Убедитесь, что вы экспортировали из **Transactions → Statements and Reports → CSV**. Старый формат «Activity Export» отличается и не поддерживается.

- **В: Перенесутся ли мои категории из Wise?**
  **О:** Категоризация самого Wise частично используется для подсказок по известным продавцам. Приложение не создаёт новые категории автоматически — если совпадения нет, строка будет импортирована без категории, и вы сможете назначить её позже.
