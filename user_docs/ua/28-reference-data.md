# Довідники

> Категорії, магазини, теги та проєкти — основа організації витрат. Керуйте всім з одного місця: **Налаштування → Довідники**.

## Огляд

Усі чотири типи довідників керуються з єдиного розділу в Налаштуваннях. Інтерфейс єдиний: **натисніть на рядок для редагування**, натисніть **+** для додавання, натисніть **іконку кошика** для видалення.

![Хаб довідників із категоріями, магазинами, тегами та проєктами](../img/reference-data.jpg)

> **Роль Спостерігач**: учасники з роллю Спостерігач можуть переглядати довідники, але не можуть додавати, перейменовувати або видаляти записи.

## Категорії

Категорії класифікують витрати та доходи. Кожна має назву та колір.

- Натисніть на рядок категорії, щоб перейменувати або змінити колір
- Натисніть **+** поруч з «Категорії витрат» або «Категорії доходів» для створення нової
- Натисніть іконку кошика для видалення категорії
  - Видалення заблоковано, якщо категорія використовується активними витратами або бюджетами
  - Системні (попередньо встановлені) категорії видалити не можна

**Порада:** Використовуйте яскраві кольори для категорій, які важливо швидко знаходити на графіках.

## Магазини

Магазини створюються автоматично при додаванні витрат. Розділ Магазини дозволяє усунути дублікати та виправити помилки.

- Натисніть на рядок магазину, щоб перейменувати його
- Перейменування об'єднує всі витрати зі старою назвою під новою (масове оновлення)
- Видалення прибирає назву магазину з усіх відповідних витрат (самі витрати зберігаються)

> Створити магазин вручну не можна — вони з'являються автоматично при додаванні витрат.

### Правила категорій

Додаток навчається на ваших виправленнях. Щоразу, коли ви змінюєте категорію витрати з назвою магазину, автоматично зберігається **правило категорії**. При наступному імпорті виписки або CSV-файлу Wise додаток застосує ваше правило і призначить категорію без ручного виправлення.

- Вивчені правила відображаються в розділі **Правила категорій** внизу екрана Магазини
- Натисніть іконку кошика, щоб видалити правило
- Правила зберігаються на сервері і синхронізуються на всіх ваших пристроях

## Теги

Теги дозволяють маркувати витрати довільними ключовими словами, незалежно від категорій.

- Натисніть **+** для створення тегу (введіть назву та виберіть колір)
- Натисніть на рядок тегу, щоб перейменувати або змінити колір
- Натисніть іконку кошика для видалення тегу з усіх витрат

**Порада:** Використовуйте теги для тимчасового відстеження: «Відрядження», «Ремонт», «Подарунки».

## Проєкти

Проєкти групують витрати за метою або діяльністю.

- Натисніть **+** для створення проєкту (назва, опис, колір, необов'язковий бюджет)
- Натисніть на рядок проєкту для відкриття детального екрану
  - Перегляньте всі пов'язані витрати, загальну суму та залишок бюджету
  - Використовуйте **іконку олівця** (вгорі справа) для редагування
  - Використовуйте **іконку кошика** (вгорі справа) для видалення
- Щоб прив'язати витрату до проєкту, відкрийте витрату та виберіть проєкт

**Порада:** Встановіть бюджет проєкту, щоб відстежувати витрати відносно мети.
