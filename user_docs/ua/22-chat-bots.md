# Чат-боти — Telegram і WhatsApp

> Керуйте фінансами прямо з Telegram або WhatsApp. Спілкуйтеся з ШІ, додавайте витрати, скануйте чеки та надсилайте голосові — без відкриття застосунку.

## Огляд

Підключіть акаунт до **Telegram**, **WhatsApp** або до обох одночасно. Обидва боти мають однакові можливості — використовуйте зручний месенджер.

Для підключення: **Налаштування → Чат-боти**.

## Прив'язка акаунту

### Telegram
1. Натисніть **Підключити Telegram** — з'явиться 6-символьний код (дійсний 10 хвилин)
2. Відкрийте Telegram і знайдіть бота
3. Надішліть `/link ВАШ_КОД` (наприклад, `/link A3F2B1`)
4. Ви побачите «Акаунт успішно прив'язано!»

### WhatsApp
1. Натисніть **Підключити WhatsApp** — з'явиться код і QR-код
2. Натисніть **Відкрити WhatsApp** (повідомлення вже заповнено) або відскануйте QR-код
3. Надішліть `link ВАШ_КОД` боту
4. Ви побачите «Акаунт успішно прив'язано!»

> Telegram і WhatsApp можна підключити одночасно до одного акаунту.

## Що можна робити

- **Додавати витрати та доходи**: пишіть природною мовою або використовуйте команди
- **ШІ-чат**: задавайте будь-які фінансові питання — той самий ШІ, що й у застосунку
- **Голосові повідомлення**: промовляйте витрату або питання (2 запити ШІ за повідомлення)
- **Фото чеків**: надішліть фото для автоматичного сканування (2 запити ШІ)
- **Перевірити використання ШІ**: `/usage`
- **Переключити акаунт**: `/account`

## Команди

| Команда | Що робить |
|---|---|
| `/link КОД` | Прив'язати месенджер до застосунку |
| `/expense 50 обід` | Додати витрату |
| `/income 3000 зарплата` | Додати дохід |
| `/category expense Їжа` | Створити категорію |
| `/usage` | Перегляд використання ШІ |
| `/account` | Переключити активний акаунт |
| `/newchat` | Почати нову розмову з ШІ |
| `/unlink` | Відключити бота |
| `/help` | Показати всі команди |

> У **WhatsApp** команди працюють як із символом `/`, так і без нього. Можна просто написати суму та опис: `50 обід`.

## Сканування чеків

1. Сфотографуйте чек і надішліть боту
2. Бот розпізнає суму, дату та продавця
3. Якщо дата невірна — надішліть правильну у форматі `ДД.ММ.РРРР`
4. Підтвердіть додавання або скасуйте

## Кілька акаунтів

- Згадайте назву акаунту в повідомленні: «Покажи витрати в Сімейному» — ШІ запитає цей акаунт лише для цього повідомлення
- Використовуйте `/account` для постійного переключення акаунту за замовчуванням

## Вартість запитів ШІ

| Дія | Запити ШІ |
|---|---|
| Текстове повідомлення / ШІ-чат | 1 |
| Голосове повідомлення | 2 |
| Фото чека | 2 |

При досягненні ліміту бот повідомить вас замість обробки запиту. Команда `/usage` покаже залишок.

## Підтримка валют

Бот розпізнає символи: ₴ (UAH), $ (USD), € (EUR), zł (PLN), £ (GBP), ₽ (RUB).

**Приклади:** `/expense 50$ обід` · `50 zł кава` · `expense 100₴ продукти`

## FAQ

**П: Чи можна підключити і Telegram, і WhatsApp?**
Так — вони прив'язуються незалежно і працюють одночасно.

**П: Якою мовою відповідає бот?**
Мовою, обраною в Налаштування → Зовнішній вигляд.

**П: Чи потрібна підписка для ботів?**
Боти використовують запити ШІ з вашого місячного ліміту. У користувачів безкоштовного плану ліміт менший.
