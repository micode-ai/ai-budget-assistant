# Імпорт транзакцій з банку

> Імпортуйте транзакції з CSV або PDF-виписки вашого банку. Підтримуються mBank, PKO BP, Erste Bank, Alior Bank, Wise та будь-який інший банк через універсальний маппер колонок.

## Підтримувані банки

- **mBank** — експорт CSV
- **PKO BP** — експорт CSV
- **Erste Bank** — PDF-виписка
- **Alior Bank** — PDF-виписка
- **Wise** — експорт CSV (мультивалютний, конвертації визначаються автоматично)
- **Інший** — будь-який банк, через універсальний маппер колонок (CSV)

## Як імпортувати

1. Перейдіть до **Налаштування → Імпорт транзакцій**
2. Виберіть ваш банк зі списку (або **Інший (CSV)**, якщо його немає)
3. Виберіть файл, експортований з банку
4. Застосунок покаже попередній перегляд — кожен рядок позначений як витрата, дохід або обмін валюти
5. Зніміть галочки з непотрібних рядків і натисніть **Імпортувати**

## Де знайти експорт у вашому банку

- **mBank**: Веб-банкінг → Historia operacji → Eksport → CSV
- **PKO BP**: iPKO → Historia operacji → Eksportuj → CSV
- **Erste Bank**: bankowość internetowa → Wyciągi → pobierz wyciąg (PDF)
- **Alior Bank**: Alior Online → Wyciągi → pobierz wyciąg (PDF)
- **Wise**: wise.com → Transactions → Statements and Reports → виберіть діапазон дат → CSV → виберіть валюту → Download

> **Порада для Wise:** Wise створює один CSV на кожен валютний баланс. Імпортуйте кожну валюту окремо. До 469 днів за один експорт.

## Wise — конвертації та комісії

При конвертації валют всередині Wise (наприклад, 100 USD → EUR) створюється два рядки. Застосунок визначає такі пари автоматично і створює один запис **Обмін валюти** (розділ Гаманець → Обміни).

Комісії Wise зі стовпця `Total fees` автоматично включаються в суму витрати.

## Що імпортується

Кожен рядок стає Витратою, Доходом або Обміном валюти. Категорії пропонуються автоматично для популярних магазинів. Повторний імпорт того самого файлу безпечний.

## «Інший» — універсальний маппер

Якщо вашого банку немає в списку, виберіть **Інший (CSV)**. Застосунок покаже попередній перегляд файлу і попросить вказати, в якому стовпці знаходяться дата, сума та опис. Збережіть маппінг для автоматичного використання надалі.

## Історія імпортів та Скасування

Розділ **Минулі імпорти** показує останні 20 імпортів. Для скасування натисніть **стрілку відміни** (↩) праворуч. Скасування доступне протягом **30 днів**.

## Не бачите свій банк?

Внизу **Налаштування → Імпорт транзакцій** є картка **«Немає вашого банку?»**. Натисніть, вкажіть назву банку та прикріпіть приклад виписки.
