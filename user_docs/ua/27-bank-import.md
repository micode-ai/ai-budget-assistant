# Імпорт транзакцій з банку

> Імпортуйте транзакції безпосередньо з CSV-експортів найбільших польських банків або будь-якого іншого банку за допомогою універсального маппера стовпців.

## Підтримувані банки

Ви можете імпортувати транзакції безпосередньо з CSV-експортів найбільших польських банків: **mBank, PKO BP, ING Bank Śląski, Bank Millennium, Pekao SA**. Для будь-якого іншого банку універсальний маппер стовпців дозволяє описати формат вручну.

## Як імпортувати

1. Перейдіть до **Налаштування → Імпорт транзакцій**
2. Виберіть свій банк зі списку (або «Інший (власний CSV)» для непідтримуваних банків)
3. Оберіть CSV-файл, експортований з інтернет-банку
4. Застосунок покаже попередній перегляд, де кожен рядок позначено як витрату, дохід або обмін валюти
5. Зніміть позначки з непотрібних рядків і натисніть **Імпортувати**

Застосунок запам'ятовує вже імпортовані рядки за датою, сумою та описом — повторне завантаження того самого CSV не створить дублікатів.

## Де знайти CSV у своєму банку

- **mBank**: Інтернет-банк → Historia operacji → Eksport → CSV
- **PKO BP**: iPKO → Lista operacji → Pobierz → CSV
- **ING Bank Śląski**: Moje ING → Historia → Eksportuj → CSV
- **Bank Millennium**: Web → Historia rachunku → Eksport → CSV
- **Pekao SA**: Pekao24 → Historia → Eksport → CSV

## Що імпортується

Кожен рядок стає Витратою, Доходом або Обміном валюти (коли застосунок виявляє парну FX-транзакцію з тією самою датою в різних валютах). Категорії пропонуються автоматично для популярних торговельних мереж (Biedronka, Żabka, Orlen, Lidl тощо) — їх можна змінити пізніше.

## «Інший» — універсальний маппер CSV

Якщо вашого банку немає у списку, виберіть «Інший (власний CSV)». Застосунок покаже попередній перегляд файлу та попросить вказати, в якому стовпці знаходяться дата, сума та опис. Ви можете зберегти це відповідність з іменем — наступний CSV з таким самим розташуванням стовпців буде імпортовано автоматично.

## Кодування

Застосунок автоматично визначає UTF-8 і Windows-1250 (найпоширеніше кодування польських банків). Якщо в попередньому перегляді польські символи відображаються неправильно, виберіть кодування вручну в маппері.
