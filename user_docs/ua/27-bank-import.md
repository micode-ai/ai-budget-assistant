# Імпорт транзакцій з банку

> Імпортуйте транзакції з CSV-файлу або PDF-виписки вашого банку, або з будь-якого іншого банку за допомогою універсального маппера стовпців.

## Підтримувані банки

- **mBank** — експорт CSV
- **PKO BP** — експорт CSV
- **Erste Bank** — PDF-виписка
- **Alior Bank** — PDF-виписка
- **Wise** — експорт CSV (див. [Імпорт з Wise](./26-wise-import.md))
- **Other** — будь-який банк, через універсальний маппер стовпців (CSV)

Підтримка нових банків додається поступово. Якщо вашого банку ще немає в списку, оберіть **Other** і вручну вкажіть відповідність стовпців.

## Як імпортувати

1. Перейдіть до **Налаштування → Імпорт транзакцій**
2. Оберіть свій банк зі списку (або **Other (custom CSV)**, якщо його немає)
3. Оберіть файл, експортований з вашого банку — **CSV** для mBank/PKO, **PDF**-виписку для Erste/Alior
4. Застосунок покаже попередній перегляд, де кожен рядок позначено як витрату, дохід або обмін валюти
5. Зніміть позначки з непотрібних рядків і натисніть **Імпортувати**

Застосунок запам'ятовує вже імпортовані рядки за датою, сумою та описом — повторне завантаження того самого файлу не створить дублікатів.

## Де знайти файл у своєму банку

- **mBank**: Інтернет-банк → Historia operacji → Eksport → CSV
- **PKO BP**: iPKO → Historia operacji → Eksportuj → CSV
- **Erste Bank**: bankowość internetowa → Wyciągi → pobierz wyciąg (PDF)
- **Alior Bank**: Alior Online → Wyciągi → pobierz wyciąg (PDF)

## Що імпортується

Кожен рядок стає Витратою, Доходом або Обміном валюти (коли застосунок виявляє парну FX-транзакцію з тією самою датою в різних валютах). Категорії пропонуються автоматично для популярних торговельних мереж (Biedronka, Żabka, Orlen, Lidl, Rossmann тощо) — їх можна змінити пізніше.

## «Other» — універсальний маппер CSV

Якщо вашого банку немає у списку, оберіть **Other (custom CSV)**. Застосунок покаже попередній перегляд файлу та попросить вказати, в якому стовпці знаходяться дата, сума та опис. Ви можете зберегти це відповідність з іменем — наступний CSV з таким самим розташуванням стовпців буде імпортовано автоматично.

## Кодування

Для CSV-файлів застосунок автоматично визначає UTF-8 і Windows-1250 (найпоширеніше кодування польських банків). Якщо в попередньому перегляді польські символи відображаються неправильно, виберіть кодування вручну в маппері. PDF-виписки зчитуються безпосередньо — вибір кодування не потрібен.
