# Налаштування

> Налаштуйте профіль, зовнішній вигляд, сповіщення та синхронізацію даних. Змінюйте мову, валюту, тему та керуйте параметрами акаунту.

## Огляд

Доступ до **Налаштування** через натискання **іконки шестерінки** у верхньому правому куті будь-якого екрана.

## Профіль

![Налаштування — розділ Профіль](../img/settings-0.jpg)

- **Аватар** — відображає ваші ініціали з кольоровим фоном
- **Ім'я** — ваше ім'я для відображення. Натисніть іконку олівця, щоб змінити
- **Ел. пошта** — email вашого акаунту (лише перегляд)
- **Часовий пояс** — натисніть іконку олівця, щоб змінити. З'являється список з можливістю пошуку серед 90+ часових поясів (наприклад, "Europe/Warsaw")

## Валюта

Оберіть вашу стандартну валюту, натиснувши один з валютних чіпів:

**USD** | **EUR** | **PLN** | **GBP** | **UAH** | **RUB** | **BYN**

Обрана валюта виділена. Це встановлює стандартну валюту для нових витрат, доходів та відображення на Головна.

## Налаштування додатку

### Мова

Оберіть з 8 підтримуваних мов:

| Код | Мова |
|---|---|
| EN | English |
| RU | Русский |
| UA | Українська |
| PL | Polski |
| ES | Espanol |
| FR | Francais |
| DE | Deutsch |
| BY | Беларуская |

Натисніть мовний чіп, щоб переключитися. Інтерфейс оновлюється миттєво.

### Стиль відповідей ШІ

Керуйте тим, як ШІ спілкується з вами в усіх функціях зі ШІ (Чат, Історії, Аналітика, Пошук зайвих витрат, Цілі):

| Режим | Опис |
|---|---|
| **Простий** | Зрозуміла мова, без жаргону — чудово для початківців |
| **Збалансований** | Поєднання простого та технічного — за замовчуванням |
| **Експертний** | Фінансова термінологія, детальний аналіз — для професіоналів |

Натисніть чіп для переключення. Зміна миттєво застосовується до всього нового контенту ШІ.

> Див. [Режим відповідей ШІ](./20-ai-response-mode.md) для деталей про кожен режим.

### Модель ШІ

Оберіть модель ШІ для всіх функцій зі ШІ (Чат, Сканування чеків, Категоризація, Аналітика, Історії, Цілі). Це безпосередньо впливає на **якість відповідей** та **витрати AI-квоти**:

| Модель | Швидкість | Вартість | Найкраще для |
|---|---|---|---|
| ⚡ **Fast** | Найшвидша | ×0.75 квоти | Швидка категоризація, прості запити |
| ⚖️ **Balanced** | Середня | ×1 квоти | Загальне використання — за замовчуванням |
| ✨ **Quality** | Повільніше | ×1.5 квоти | Складний аналіз, детальні інсайти |

> **Порада:** При тарифі Free (5 AI запитів/місяць) переключення на **Fast** дає ~6.7 ефективних запити. Переключення на **Quality** — ~3.3.

Натисніть чіп для зміни. Модель застосовується миттєво до всіх нових AI-запитів.

### Зовнішній вигляд

Оберіть тему:

- **Системна** — слідує налаштуванню світлого/темного режиму вашого пристрою
- **Світла** — завжди використовувати світлу тему
- **Темна** — завжди використовувати темну тему

## Віджети головного екрану

Керуйте тим, які секції відображаються на [Головному екрані](./02-dashboard.md). Натисніть на перемикач поряд з віджетом, щоб показати або приховати його:

| Віджет | Опис |
|---|---|
| **Гейміфікація** | Рівень, смуга прогресу досвіду та щоденна серія |
| **Місячний бюджет** | Залишок бюджету з кольоровою смугою прогресу |
| **Доходи та витрати** | Об'єднана картка з доходами та витратами за місяць |
| **Борги та позики** | Зведення того, хто вам повинен і що повинні ви |
| **Графік чистого прибутку** | Лінійний графік чистого прибутку за 6 місяців |
| **Чистий капітал** | Загальний стан по всіх гаманцях |
| **Fat Finder (ШІ-аналітика)** | Щомісячний аудит витрат з можливостями заощадження |
| **Календар** | Місячна сітка з точками доходів та витрат |
| **Цілі** | Головна активна ціль заощаджень з прогрес-баром |
| **Баланси гаманців** | Горизонтальне прокручування балансів за валютами |

> **Порада:** Ваші налаштування зберігаються автоматично і не скидаються при перезапуску.

## Сповіщення

![Налаштування — розділ Сповіщення](../img/settings.jpg)

Перемикачі для налаштування сповіщень:

| Параметр | Опис |
|---|---|
| **Push-сповіщення** | Головний перемикач — увімкнути або вимкнути всі push-сповіщення |
| **Сповіщення про бюджет** | Отримувати сповіщення при досягненні порогів бюджету |
| **Активність спільного рахунку** | Отримувати сповіщення, коли інші додають витрати до спільних рахунків |

## Підписка

- **Керування тарифом** — натисніть, щоб переглянути поточну підписку та дослідити варіанти оновлення

## Панель адміністратора

- **Панель адміністратора** — видима лише для адміністраторів. Надає статистику рівня системи та моніторинг використання ШІ.

## Гаманець

Швидкий доступ до функцій гаманця:

- **Встановити баланс** — встановити початкові баланси для ваших валют
- **Баланси** — переглянути детальну розбивку гаманця за валютами

## Звіти та email

- **Щотижнева email-зведення** (Business) — увімкнення/вимкнення, вибір дня тижня
- **Щомісячний дайджест** (Pro+) — увімкнення/вимкнення
- **Згенерувати звіт** — відкриває екран [Експорт і звіти](./16-export-reports.md)
- **Створити резервну копію** — створює повну JSON-копію даних акаунту
- **Імпортувати резервну копію** — завантаження та відновлення даних із раніше створеної копії

## Дані та синхронізація

- **Остання синхронізація** — показує, коли ваші дані були останній раз синхронізовані з сервером (наприклад, "5 хв. тому" або "Ніколи")
- **Синхронізувати** — натисніть, щоб вручну запустити синхронізацію даних

> **Примітка:** Додаток працює офлайн. Ваші дані зберігаються на пристрої та автоматично синхронізуються при підключенні до мережі. Використовуйте **Синхронізувати**, щоб примусово запустити негайну синхронізацію.

## Вихід

Прокрутіть донизу в Налаштування та натисніть **Вийти**. З'явиться діалог підтвердження — підтвердіть, щоб вийти з акаунту.

## Часті запитання

- **П:** Я змінив мову, але деякий текст все ще на старій мові. Що робити?
  **В:** Зміна мови миттєва для всіх елементів інтерфейсу. Якщо ви помітили неперекладений текст, спробуйте перезапустити додаток.

- **П:** Як змінити email?
  **В:** Зміна email наразі не підтримується в додатку. Зверніться до служби підтримки для допомоги.

- **П:** Що станеться з моїми даними при виході?
  **В:** Ваші дані залишаються на сервері. При повторному вході все буде відновлено. Локальні дані на пристрої можуть бути очищені.
