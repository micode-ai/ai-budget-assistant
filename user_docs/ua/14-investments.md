# Інвестиційний портфель

> Відстежуйте інвестиційний портфель з актуальними ринковими цінами. Моніторте акції, ETF, криптовалюти, облігації та товари в одному місці.

## Огляд

Функція інвестиційного портфеля дозволяє:

- Відстежувати позиції за різними типами активів
- Бачити ціни в реальному часі та вартість портфеля
- Аналізувати дохідність за різні періоди
- Порівнювати свої результати з ринковими бенчмарками (SPY, QQQ, DIA, IWM)
- Записувати угоди купівлі/продажу з комісіями

## Створення інвестиційного рахунку

Для відстеження інвестицій потрібен спеціальний тип рахунку **Інвестиції**:

1. Перейдіть на вкладку **Рахунки**
2. Натисніть **Створити рахунок**
3. Виберіть тип **Інвестиції**
4. Назвіть портфель (наприклад, "Основний портфель", "Пенсійний")
5. Натисніть **Створити**

## Додавання позицій

### Пошук активів

1. Відкрийте інвестиційний рахунок
2. Натисніть **Додати позицію**
3. Шукайте за тікером (наприклад, "AAPL") або назвою компанії (наприклад, "Apple")
4. Виберіть потрібний актив з результатів пошуку
5. Додайте нотатки (опціонально)
6. Натисніть **Зберегти**

### Підтримувані типи активів

| Тип | Приклади |
|-----|----------|
| Акції | AAPL, MSFT, GOOGL |
| ETF | SPY, QQQ, VTI |
| Криптовалюти | BTC, ETH, SOL |
| Облігації | Державні та корпоративні |
| Товари | Золото, срібло, нафта |

## Запис транзакцій

Після додавання позиції записуйте угоди купівлі/продажу:

1. Відкрийте деталі позиції
2. Натисніть **Додати транзакцію**
3. Виберіть тип: **Купівля** або **Продаж**
4. Введіть:
   - **Кількість** — число акцій/одиниць
   - **Ціна за одиницю** — ціна купівлі/продажу
   - **Комісія** — брокерська комісія (опціонально)
   - **Дата** — дата угоди
   - **Нотатки** — додаткова інформація (опціонально)
5. Натисніть **Зберегти**

Додаток автоматично розраховує:
- **Середня ціна купівлі** — середньозважена ціна придбання
- **Сума інвестицій** — сума всіх покупок мінус продажі
- **Поточний прибуток/збиток** — на основі поточної ціни

## Зведення по портфелю

Головний екран інвестицій показує:

- **Загальна вартість** — поточна ринкова вартість всіх позицій
- **Загальний прибуток/збиток** — сума прибутку або збитку
- **Загальна дохідність %** — відсотковий дохід
- **Зміна за день** — сьогоднішня зміна вартості

Для кожної позиції відображається:
- Поточна ціна та денна зміна
- Ваша кількість та середня ціна
- Індивідуальний прибуток/збиток та частка в портфелі

## Аналітика

Доступ до детальної аналітики портфеля:

1. Натисніть кнопку **Аналітика**
2. Виберіть період: 1Т, 1М, 3М, 1Р або Весь час

### Графік продуктивності

Показує вартість портфеля в часі порівняно з інвестованою сумою. Область між лініями відображає ваш прибуток або збиток.

### Розподіл за типами

Візуалізує розподіл портфеля за типами активів (акції, ETF, крипто тощо).

### Лідери зростання та падіння

Список найкращих та найгірших позицій за відсотковою дохідністю.

### AI-інсайти портфеля (Pro+)

Отримуйте AI-аналіз вашого портфеля з рекомендаціями до дії:

1. Відкрийте вкладку **Аналітика**
2. Прокрутіть до каруселі **Інсайти** вгорі
3. Свайпайте вліво/вправо для перегляду різних інсайтів
4. Натисніть кнопку закриття, щоб приховати інсайт

**Типи інсайтів:**

| Тип | Опис |
|-----|------|
| Ризик концентрації | Попереджає, коли один актив домінує в портфелі |
| Дисбаланс секторів | Сигналізує про перевагу одного типу активів |
| Відстаючий | Виявляє активи, що відстають від ринку |
| Лідер зростання | Підсвічує можливості для ребалансування |
| Відхилення від бенчмарка | Показує відхилення портфеля від бенчмарка |
| Прогалина диверсифікації | Пропонує додати відсутні типи активів |
| Податковий сигнал | Повідомляє про суттєві нереалізовані П/З |
| Вплив комісій | Попереджає, коли комісії з'їдають прибуток |

Кожен інсайт включає візуалізацію (графік) та рекомендацію до дії.

**Примітка:** AI-інсайти кешуються на 24 години, вартість оновлення — 2.5 AI-кредитів.

### Порівняння з бенчмарком (Pro+)

Порівняйте дохідність портфеля з ринковими індексами:

| Бенчмарк | Опис |
|----------|------|
| SPY | Індекс S&P 500 |
| QQQ | Індекс Nasdaq 100 |
| DIA | Індекс Dow Jones Industrial |
| IWM | Russell 2000 (мала капіталізація) |

**Розуміння порівняння:**
- **Дохідність портфеля** — ваш фактичний прибуток/збиток у відсотках
- **Дохідність бенчмарка** — продуктивність індексу за той самий період
- **Різниця** — наскільки ви перевершили або відстали від ринку

## Розуміння розрахунків

Натисніть на будь-яку картку аналітики, щоб побачити пояснення формули:

### Дохідність
```
Дохідність % = ((Кінцева вартість - Початкова вартість) / Початкова вартість) x 100
```

### Прибуток/збиток (П/З)
```
П/З = Поточна вартість - Сума інвестицій
П/З % = (П/З / Сума інвестицій) x 100
```

### Розподіл
```
Частка % = (Вартість активу / Загальна вартість портфеля) x 100
```

## Оновлення цін

- Ціни оновлюються автоматично кожні 15 хвилин
- Натисніть кнопку **Оновити** для негайного оновлення
- Історичні ціни кешуються для економії трафіку

## Поради

1. **Диверсифікуйте облік** — додайте всі інвестиції для повної картини
2. **Враховуйте комісії** — включайте брокерські збори для точного розрахунку прибутку
3. **Використовуйте бенчмарки** — порівнюйте з індексами для оцінки результатів
4. **Регулярно перевіряйте** — переглядайте аналітику щотижня для виявлення трендів

## Обмеження

- Дані про ціни надаються через Twelve Data API
- Деякі екзотичні інструменти можуть бути недоступні
- Історичні дані обмежені торговими днями
- Ціни в реальному часі можуть мати затримку до 15 хвилин
