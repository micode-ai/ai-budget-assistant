# AI Budget Assistant

Приложение для управления личными финансами с искусственным интеллектом: голосовой ввод, сканирование чеков и интеллектуальная категоризация расходов.

## Обзор

AI Budget Assistant — кроссплатформенное мобильное приложение, которое помогает пользователям отслеживать расходы, управлять бюджетами и получать аналитику по финансовым привычкам с использованием искусственного интеллекта.

### Основные возможности

- **Учёт расходов** — ручной ввод, голосовой ввод и сканирование чеков
- **Учёт доходов** — запись доходов с категориями, мультивалютная поддержка, разбивка по валютам на главной панели
- **Позиции расходов** — детализация чеков с позициями (количество, цена за единицу)
- **AI-категоризация** — автоматическое распределение расходов по категориям с помощью GPT-4
- **Голосовой ввод** — транскрипция расходов через Whisper API
- **Сканирование чеков** — извлечение данных из чеков с помощью GPT-4 Vision
- **Управление бюджетами** — создание и отслеживание бюджетов с оповещениями
- **Мультиаккаунтность** — личные, бизнес и общие аккаунты с ролевым доступом (owner/editor/viewer)
- **Кошелёк и обмен валют** — мультивалютные балансы и отслеживание курсов обмена
- **Теги** — гибкая система тегов для расходов и доходов с цветовой маркировкой, AI-подсказки тегов
- **Проекты** — группировка расходов и доходов по проектам с бюджетами, сроками и аналитикой по проекту
- **Разделение расходов** — разделение одного расхода по нескольким категориям с суммами и процентами, AI-подсказки разделения
- **AI-чат ассистент** — персонализированные финансовые советы
- **Умная аналитика** — обнаружение аномалий расходов и прогнозы исчерпания бюджетов
- **Push-уведомления** — оповещения о бюджетах и аномалиях расходов (Expo Push)
- **Общие аккаунты** — приглашение участников по инвайт-кодам, отслеживание активности
- **Аналитика** — визуализация трендов, распределение по категориям, тегам и проектам
- **Интерактивные графики** — детализация от года до отдельных транзакций с анимированными столбчатыми, линейными и кольцевыми графиками
- **AI Инсайты** — GPT-4 анализирует паттерны расходов и генерирует визуальные карточки с рекомендациями (Pro)
- **История расходов** — AI-генерируемые нарративные дашборды с интерактивными блоками, резюмирующие финансовый период (Pro)
- **Виджеты домашнего экрана** — Android-виджеты 3 размеров: итоги за день, недельные графики, прогресс бюджетов
- **Система подписок** — уровни Free, Pro и Business с настраиваемыми лимитами AI запросов
- **Панель администратора** — системная статистика, разбивка подписок, мониторинг AI-использования по пользователям
- **Offline-first** — полная функциональность без интернета
- **Кроссплатформенность** — iOS, Android и Web

## Технологический стек

| Слой | Технология |
|------|------------|
| Мобильное приложение | React Native / Expo 50 |
| Управление состоянием | Zustand |
| Получение данных | TanStack React Query |
| Локальная БД | SQLite + Drizzle ORM |
| Бэкенд | NestJS 10 |
| База данных | PostgreSQL + Prisma |
| Кэш | Redis |
| Аутентификация | JWT + Passport |
| AI/ML | OpenAI (GPT-4, Whisper, Vision) |
| Графики | react-native-gifted-charts |
| Виджеты | react-native-android-widget |
| Система сборки | Turborepo |
| Пакетный менеджер | npm |
