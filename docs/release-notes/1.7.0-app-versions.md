# Release notes for admin → App Versions — 1.7.0

Store / in-app `UpdatePrompt` release notes for version **1.7.0**, one block per
locale. Paste each into the matching field in the admin **App Versions** form
(EN is required; the other 8 are optional). Locales: `en`, `ru`, `ua`, `pl`,
`es`, `fr`, `de`, `be`, `nl`.

---

## en

```
New in 1.7.0:
• Wallet balance chart — see your monthly balance change with a 6/12-month toggle and a currency switch.
• Subscription renewal calendar — switch the Subscriptions screen between List and Calendar views.
• Cleaner merchant names from bank imports, plus bulk-merge and smart grouping suggestions on the Merchants screen.
• Fixes: Slack file uploads (photos, voice, PDF) work again, and summary report emails were restyled.
```
