# Release notes for admin → App Versions — 1.10.0

Store / in-app `UpdatePrompt` release notes for version **1.10.0**, one block per
locale. Paste each into the matching field in the admin **App Versions** form
(EN is required; the other 8 are optional). Locales: `en`, `ru`, `ua`, `pl`,
`es`, `fr`, `de`, `be`, `nl`.

---

## en

```
New in 1.10.0:
• Purchase Requests — propose group purchases on shared accounts; members vote Approve/Reject via app or bots; approval creates a planned expense.
• Family Feed — Instagram-style activity feed for shared accounts with emoji reactions. New home widget.
• Fix: Bank notification capture improvements for PKO BP and React Native New Architecture.
```
