# Release notes for admin → App Versions — 1.9.0

Store / in-app `UpdatePrompt` release notes for version **1.9.0**, one block per
locale. Paste each into the matching field in the admin **App Versions** form
(EN is required; the other 8 are optional). Locales: `en`, `ru`, `ua`, `pl`,
`es`, `fr`, `de`, `be`, `nl`.

---

## en

```
New in 1.9.0:
• Safe-to-Spend — new home card shows how much you can spend this month; ask the AI "can I afford X?" for an instant answer.
• Bank notification capture (Android) — app reads bank push notifications and suggests adding expenses automatically. Enable in Settings → Auto-capture.
• Smart dedup — captured notifications that match an imported CSV transaction are flagged as duplicates with a merge option.
• Desktop web layout — sidebar navigation, top bar, and two-column Dashboard grid at 1024 px+.
• Tracking gap reminder — nudge if you haven't logged expenses for 3+ days. Settings → Notifications.
• Fix: Sign in with Google now works on the web app.
```
