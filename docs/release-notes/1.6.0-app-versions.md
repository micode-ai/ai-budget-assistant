# Release notes for admin → App Versions — 1.6.0

Store / in-app `UpdatePrompt` release notes for version **1.6.0**, one block per
locale. Paste each into the matching field in the admin **App Versions** form
(EN is required; the other 8 are optional). Locales: `en`, `ru`, `ua`, `pl`,
`es`, `fr`, `de`, `be`, `nl`.

---

## en

```
New in 1.6.0:
• Anomaly alerts — the app now flags unusual activity: possible duplicate charges, price increases, "looks like a subscription" suggestions, and category spikes. Tap the new bell on the home screen.
• Cleaner home & header — account and display currency are now separate pills.
• Quick actions wrap to rows — every shortcut visible at once, with refreshed icons.
• Fixes: comma as decimal separator, web sign-out, and Belarusian translations.
```
