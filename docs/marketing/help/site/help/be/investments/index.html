Інвестыцыйны партфель

Адсочвайце інвестыцыйны партфель з актуальнымі рынкавымі цэнамі. Маніторце акцыі, ETF, крыптавалюты, абліга цыі і тавары ў адным месцы.

Агляд

Функцыя інвестыцыйнага партфеля дазваляе:

  • Адсочваць пазіцыі па розных тыпах актываў
  • Бачыць цэны ў рэальным часе і кошт партфеля
  • Аналізаваць прыбытковасць за розныя перыяды
  • Параўноўваць свае вынікі з рынкавымі бенчмаркамі (SPY, QQQ, DIA, IWM)
  • Запісваць здзелкі купля/продажу з камісіямі

Стварэнне інвестыцыйнага рахунку

Для адсочвання інвестыцый патрэбны спецыяльны тып рахунку Інвестыцыі:

  1. Перайдзіце на ўкладку Рахункі
  2. Націсніце Стварыць рахунак
  3. Выберыце тып Інвестыцыі
  4. Назавіце партфель (напрыклад, “Асноўны партфель”, “Пенсійны”)
  5. Націсніце Стварыць

Даданне пазіцый

Пошук актываў

  1. Адкрыйце інвестыцыйны рахунак
  2. Націсніце Дадаць пазіцыю
  3. Шукайце па тыкеру (напрыклад, “AAPL”) або назве кампаніі (напрыклад, “Apple”)
  4. Выберыце патрэбны актыў з вынікаў пошуку
  5. Дадайце нататкі (апцыянальна)
  6. Націсніце Захаваць

Падтрымліваемыя тыпы актываў

Тып Прыклады
Акцыі AAPL, MSFT, GOOGL
ETF SPY, QQQ, VTI
Крыптавалюты BTC, ETH, SOL
Аблігацыі Дзяржаўныя і карпаратыўныя
Тавары Золата, срэбра, нафта

Запіс транзакцый

Пасля дадання пазіцыі запісвайце здзелкі купля/продажу:

  1. Адкрыйце дэталі пазіцыі
  2. Націсніце Дадаць транзакцыю
  3. Выберыце тып: Купля або Продаж
  4. Увядзіце: - Колькасць — колькасць акцый/адзінак - Цана за адзінку — цана куплі/продажу - Камісія — брокерская камісія (апцыянальна) - Дата — дата здзелкі - Нататкі — дадатковая інфармацыя (апцыянальна)
  5. Націсніце Захаваць

Дадатак аўтаматычна разлічвае: - Сярэдняя цана куплі — сярэднеўзважаная цана набыцця - Сума інвестыцый — сума ўсіх куплям мінус продажу - Бягучы прыбытак/страта — на аснове бягучай цаны

Зводка па партфелі

Галоўны экран інвестыцый паказвае:

  • Агульны кошт — бягучы рынкавы кошт усіх пазіцый
  • Агульны прыбытак/страта — сума прыбытку або страты
  • Агульная прыбытковасць % — адсоткавы прыбытак
  • Змена за дзень — сённяшняя змена кошту

Для кожнай пазіцыі адлюстроўваецца: - Бягучая цана і дзённая змена - Ваша колькасць і сярэдняя цана - Індывідуальны прыбытак/страта і доля ў партфелі

Аналітыка

Доступ да падрабязнай аналітыкі партфеля:

  1. Націсніце кнопку Аналітыка
  2. Выберыце перыяд: 1Т, 1М, 3М, 1Г або Увесь час

Графік прадукцыйнасці

Паказвае кошт партфеля ў часе ў параўнанні з інвеставанай сумай. Вобласць паміж лініямі адлюстроўвае ваш прыбытак або страту.

Размеркаванне па тыпах

Візуалізуе размеркаванне партфеля па тыпах актываў (акцыі, ETF, крыпта і г.д.).

Лідэры росту і падзення

Спіс лепшых і горшых пазіцый па адсоткавай прыбытковасці.

Разуменне разлікаў

Націсніце на любую картку аналітыкі, каб убачыць тлумачэнне формулы:

Прыбытковасць

Прыбытковасць % = ((Канчатковы кошт - Пачатковы кошт) / Пачатковы кошт) x 100

Прыбытак/страта (П/С)

П/С = Бягучы кошт - Сума інвестыцый
П/С % = (П/С / Сума інвестыцый) x 100

Размеркаванне

Доля % = (Кошт актыва / Агульны кошт партфеля) x 100

Абнаўленне цэн

  • Цэны абнаўляюцца аўтаматычна кожныя 15 хвілін
  • Націсніце кнопку Абнавіць для неадкладнага абнаўлення
  • Гістарычныя цэны кэшуюцца для эканоміі трафіку

Парады

  1. Дыверсіфікуйце ўлік — дадайце ўсе інвестыцыі для поўнай карціны
  2. Улічвайце камісіі — уключайце брокерскія зборы для дакладнага разліку прыбытку
  3. Выкарыстоўвайце бенчмаркі — параўноўвайце з індэксамі для ацэнкі вынікаў
  4. Рэгулярна правярайце — праглядайце аналітыку штотыдзень для выяўлення трэндаў

Абмежаванні

  • Даныя пра цэны падаюцца праз Twelve Data API
  • Некаторыя экзатычныя інструменты могуць быць недаступныя
  • Гістарычныя даныя абмежаваныя гандлёвымі днямі
  • Цэны ў рэальным часе могуць мець затрымку да 15 хвілін
